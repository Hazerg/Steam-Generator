HOW TO DOWNLOAD

1. TURN OFF ANY AV this will prevent you from downloading and using the generator
2. First download it as .zip
3. Then extract it just using the windows extractor
4.  And at least run the .exe and choose your modes 
5.  Have Fun!!!!
